# Governance

Translation status: draft

## Governance model

This project follows SaaSCoop governance principles.

## Roles

- Supporter
- Community member
- Contributor
- Maintainer
- Future cooperative member

## Decisions

Relevant decisions should be documented through:

- Issues
- Discussions
- Pull Requests
- ADRs
- Roadmap updates

## Cautions

This project must avoid conflicts of interest, must not depend on public-sector operation and must not present SaaSCoop as a formally incorporated cooperative before legal validation.
