# Cómo Participar

Translation status: draft

## Ways to participate

You can contribute through:

- Code
- Documentation
- Translation
- Testing
- Research
- Design
- Community support
- Educational content

## First steps

1. Read the README.
2. Review open issues.
3. Join discussions when available.
4. Suggest improvements.
5. Open focused pull requests.

## Community principle

Participation does not automatically imply employment, ownership or cooperative membership.
