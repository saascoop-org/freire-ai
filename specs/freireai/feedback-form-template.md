# FreireAI Feedback Form Template

Use this template to create the external feedback form for invited validation in PT-BR, EN, and ES. The simplest setup is one form with three sections, one per language, or three separate forms linked from the submission/demo notes.

## Live Form Links

- PT-BR: https://forms.gle/DqN81THFrxWgRTM29
- EN: https://forms.gle/Nh8DMk3V8YLgks9o7
- ES: https://forms.gle/7rhoc6rcBh1LCTxt7

## PT-BR

### Título do formulário

FreireAI - Feedback da Demo Piloto

### Descrição do formulário

Obrigado por testar o FreireAI. Esta demo é apenas para validação por convite. Não inclua dados sensíveis de estudantes. Use exemplos fictícios ou informações mínimas ao descrever sua experiência.

### Perguntas

1. Qual é o seu perfil?
   - Educador(a)
   - Gestor(a)/coordenador(a)
   - Pesquisador(a)
   - Estudante
   - Outro

2. A demo foi útil para imaginar uma prática real de alfabetização?
   - Escala linear de 1 a 5

3. Em que momento o FreireAI mais ajudou?
   - Resposta longa

4. Em que momento a interface ficou confusa ou difícil?
   - Resposta longa

5. As palavras geradoras e temas sugeridos fizeram sentido para o contexto do estudante?
   - Sim
   - Parcialmente
   - Não
   - Não tenho certeza

6. A codificação gerada ajudaria uma conversa real em círculo de cultura?
   - Sim
   - Parcialmente
   - Não
   - Não tenho certeza

7. O assistente de perguntas preservou o papel do educador e evitou dar respostas prontas?
   - Sim
   - Parcialmente
   - Não
   - Não tenho certeza

8. O que deveria mudar antes de testar com estudantes reais?
   - Resposta longa

9. Contato opcional para retorno
   - Resposta curta

## EN

### Form Title

FreireAI - Pilot Demo Feedback

### Form Description

Thank you for testing FreireAI. This demo is for invited validation only. Please do not include sensitive learner data. Use fictional examples or minimal information when describing your experience.

### Questions

1. What is your profile?
   - Educator
   - Manager/coordinator
   - Researcher
   - Student
   - Other

2. Was the demo useful for imagining a real literacy practice?
   - Linear scale from 1 to 5

3. At what moment did FreireAI help the most?
   - Long answer

4. At what moment did the interface feel confusing or difficult?
   - Long answer

5. Did the suggested generative words and themes make sense for the learner context?
   - Yes
   - Partially
   - No
   - Not sure

6. Would the generated codification support a real culture-circle conversation?
   - Yes
   - Partially
   - No
   - Not sure

7. Did the question assistant preserve the educator's role and avoid giving final answers?
   - Yes
   - Partially
   - No
   - Not sure

8. What should change before testing with real learners?
   - Long answer

9. Optional contact for follow-up
   - Short answer

## URL Configuration

The live form URLs are listed above. Include them in the deployment notes or submission package.

If you later add a visible feedback button to the app, use an environment variable such as:

```env
FEEDBACK_FORM_URL_PT_BR=https://forms.gle/DqN81THFrxWgRTM29
FEEDBACK_FORM_URL_EN=https://forms.gle/Nh8DMk3V8YLgks9o7
FEEDBACK_FORM_URL_ES=https://forms.gle/7rhoc6rcBh1LCTxt7
```
